# 指南

## 介绍

::: tip
开箱即用的 Vue 组件库
:::

## Install

```bash
npm install vtp
# or
yarn add vtp
```

## Used

### 一键全局引入

```javascript
import Vue from 'vue'
import Vtp from 'vtp'

Vue.use(Vtp)
```

### 按需引入

```javascript
import Vue from 'vue'
import { VtpButton } from 'vtp'

Vue.use(VtpButton)
```
